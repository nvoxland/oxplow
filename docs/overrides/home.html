// what oxplow gives you

Streams + worktrees

Each stream owns a branch and an isolated checkout. Agents on different streams cannot clobber each other.

Durable work queue

Tasks with a real lifecycle — ready, in_progress, human_check, done — grouped by epic, ordered by drag.

Commit & wait points

Inline gates in the queue. Commit when the agent reaches a milestone; pause until you release the gate.

Local History

Every turn snapshots the files it touched. Browse efforts per work item, jump back to any end snapshot.

Editor + blame + diff

Monaco at the core, with git decorations, blame overlay, and per-effort diff tabs built in.

Agent control plane

An MCP surface and Claude Code hook integration that auto-progresses the agent through your queue.
